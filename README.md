# Employee Management System

A Java-based Employee Management System developed using Oracle SQL and JDBC.

## Features

- Add Employee
- Update Employee
- Delete Employee
- Search Employee
- View Employee Details
- Department Management
- Salary Management
- Report Generation
- Exception Handling
- Input Validation

## Technologies Used

- Java
- Oracle SQL
- JDBC
- OOP
- Git & GitHub

## Project Structure

src/

sql/
